## 📦 各模块详细优化方案

### 1. AI数字分身
- **优化内容**：
  - 5位学科导师：数学老师、语文老师、英语老师、物理老师、化学老师
  - 语音输入功能：webkitSpeechRecognition
  - 对话功能：sendAvatarChat()
  - 拍照上传：handleAvatarPhoto()

### 2. AI精准练
- **优化内容**：
  - 拍照上传错题功能
  - AI分析薄弱点
  - 动态生成练习题
  - 正确率统计

### 3. 母题训练
- **优化内容**：
  - 从126道增加到378道
  - 按年级分类：五年级、六年级、初一、初二、初三
  - 按科目分类：语文、数学、英语、物理、化学
  - AI批改功能

### 4. 学霸方法
- **优化内容**：
  - 从5个增加到9个方法
  - 新增：刻意练习法、SQ3R阅读法、间隔重复法、主动回忆法
  - 每个方法添加5道训练题，共45题

### 5. 播客课堂
- **优化内容**：
  - 从13个增加到21个播客
  - 添加url和type字段
  - 支持音频和视频播放
  - 分类：method、knowledge、focus、share

### 6. 视频课堂
- **优化内容**：
  - 添加本地视频上传功能
  - 支持多格式视频播放
  - 视频播放器弹窗

### 7. 训练游戏
- **优化内容**：
  - 8种游戏全部实现
  - 独占满屏显示
  - 难度自适应（根据用户难度级别）
  - 返回按钮和重新开始功能

### 8. 学习计划
- **优化内容**：
  - Week1-4已有内容
  - 添加Week5详细内容：系统性思维与守恒思维综合周
  - 添加Week6详细内容：学科深度整合与自主学习能力培养周
  - 每周7天详细训练内容

### 9. 设置页面
- **优化内容**：
  - API Key配置
  - 难度级别调整（Lv.1-5）
  - DeepSeek充值按钮
  - 清除所有数据按钮
  - 关于系统弹窗
  - 系统更新功能

## 🔧 技术实现

### 文件结构
```
cognitive-training-portal/
├── index.html              # 主文件 (V86)
├── index_v86_backup.html   # 备份文件
├── RELEASE_NOTES_V86.md    # 发布说明
├── 优化方案汇总_V86.md      # 本文件
├── manifest.json           # PWA配置
├── service-worker.js       # 服务工作者
├── images/                 # 图片资源
├── videos/                 # 视频资源
└── podcasts/               # 播客脚本
```

### 关键函数
- `showPage()`: 页面切换
- `loadData()/saveData()`: 数据存储
- `startGame()`: 游戏启动
- `startPractice()`: 精准练开始
- `selectLevel()`: 难度选择
- `toggleTask()`: 任务展开/收起
