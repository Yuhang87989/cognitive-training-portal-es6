# 播客脚本

哈喽各位同学，你是不是有时候做数学题的时候，盯着题目看半天，感觉每个字都认识，拼到一起就不知道要干嘛？或者老师讲的时候好像都听懂了，自己一做就卡壳？其实啊，很多时候不是你不够聪明，是你还没摸清楚数学里最核心的两个武功招式——抽象思维和逻辑推理。今天咱们就好好聊聊这两个东西，看完这一期，以后再遇到数学题，你可能就会像开了天眼一样，一眼看透它的本质。

那先来说说什么是抽象思维吧，你可以把它想象成一个超级过滤器。就比如你去买奶茶，店员跟你说第二杯半价，你肯定不会想“为什么第二杯要半价”或者“奶茶里有多少珍珠”这些没用的，你只会算“买两杯的话，总共要花多少钱”。这其实就是在做抽象了——你把奶茶、珍珠、店员这些乱七八糟的细节都过滤掉了，只留下最核心的数学关系：原价乘以1.5就是两杯的总价。

再举个数学题的例子，你肯定做过那种行程问题，什么甲乙两人从AB两地相向而行，甲的速度是多少，乙的速度是多少，问多久相遇。你要是真去纠结甲长什么样子，乙穿什么衣服，那这题你一辈子都做不出来。抽象思维就是让你把这些没用的细节都扔掉，只留下“速度、时间、路程”这三个核心要素，然后用“路程=速度×时间”这个公式把它们串起来。你看，这么一想是不是就简单多了？

其实抽象思维不只是在数学里有用，生活里到处都是。比如你玩王者荣耀的时候，你不用知道每个英雄的背景故事有多复杂，你只需要知道他的技能冷却时间、伤害数值、移动速度这些核心数据，然后根据这些来决定什么时候开团什么时候撤退。这也是抽象思维啊——把复杂的角色简化成几个关键参数，然后用这些参数来制定策略。

好，说完抽象思维，咱们再来说说逻辑推理。如果说抽象思维是把复杂问题变简单的过滤器，那逻辑推理就是帮你一步一步找到答案的导航仪。逻辑推理主要有三种方式，演绎、归纳和类比。

先来说演绎推理，这个其实是我们最常用的，就是从一般到特殊。比如说，你知道所有的猫都会抓老鼠，然后你看到一只三花猫，你就可以推断出这只三花猫也会抓老鼠。再比如数学里，你知道三角形的内角和是180度，当你遇到一个直角三角形的时候，你就可以直接用这个结论，算出另外两个角的和是90度。演绎推理就像搭积木一样，从一个确定的大前提，一步一步推导出小结论，每一步都是严丝合缝的。

然后是归纳推理，这个是从特殊到一般。比如说你看到的第一只天鹅是白的，第二只也是白的，第三只还是白的，你就会归纳出“所有天鹅都是白的”这个结论。当然，后来人们发现了黑天鹅，这个结论就被推翻了，但这也恰恰说明归纳推理是需要不断验证的。在数学里，归纳推理也很常用，比如你算1+3=4，1+3+5=9，1+3+5+7=16，你就会发现结果都是平方数，然后归纳出从1开始的连续奇数相加，和等于奇数个数的平方。这个规律就是你通过归纳发现的，然后你再用演绎推理去证明它是不是对的。

最后是类比推理，这个是从相似到相似。比如说你学会了整数的加减乘除，再学分数的时候，就可以类比整数的运算规则。整数里的交换律、结合律，在分数里同样适用，只是多了一个通分的步骤。再比如你学过了平面几何里的三角形面积公式，当你学立体几何里的三棱锥体积的时候，就可以类比三角形面积公式的推导过程，用底面积乘以高再除以3来计算。类比推理就像搭桥一样，把你已经掌握的知识和新的知识连接起来，让你学新东西的时候更快上手。

光说不练假把式，咱们现在就来做一道综合题，看看怎么用抽象思维和逻辑推理来解决它。题目是这样的：小明去买笔，买了3支钢笔和2支圆珠笔，一共花了32元；小红买了2支钢笔和3支圆珠笔，一共花了28元。问钢笔和圆珠笔的单价各是多少？

你拿到这道题的时候，首先要用抽象思维把它简化。这里面的核心信息就是“钢笔单价×3 + 圆珠笔单价×2 = 32”和“钢笔单价×2 + 圆珠笔单价×3 = 28”，其他的小明、小红这些都是没用的细节，可以直接忽略。然后你可以设钢笔的单价是x元，圆珠笔的单价是y元，把这两个关系写成方程组：3x + 2y = 32，2x + 3y = 28。

接下来就用逻辑推理来解这个方程组。你可以把两个方程加起来，得到5x + 5y = 60，两边都除以5，就得到x + y = 12。也就是说，一支钢笔加一支圆珠笔要12元。然后你把这个结果代入第一个方程，3x + 2y = 32可以写成2(x + y) + x = 32，也就是2×12 + x = 32，算出来x=8，然后y=12-8=4。所以钢笔的单价是8元，圆珠笔是4元。你看，是不是很简单？

那平时怎么训练这两种思维能力呢？其实也不用刻意去做什么难题，生活里到处都是训练的机会。比如你去超市买东西的时候，可以自己先算一下总价，看看和收银员算的一样不一样，这个过程就是在训练抽象思维——把商品的价格和数量抽象成数学运算。再比如你看侦探小说的时候，可以跟着侦探一起推理凶手是谁，这个过程就是在训练逻辑推理——从线索一步一步推导出结论。

在数学学习里，你可以多做一些应用题，不是为了刷题，而是为了锻炼自己把实际问题抽象成数学模型的能力。做完题之后，不要只看答案对不对，要想想自己是怎么想到这个方法的，有没有更简单的方法。平时也可以多和同学讨论题目，看看别人是怎么思考的，说不定别人的思路会给你带来新的启发。

最后我再总结一下，抽象思维帮你把复杂问题变简单，找到问题的核心；逻辑推理帮你一步一步找到答案，让你的思考更有条理。这两个能力就像数学王国里的左右护法，只要你把它们练好了，不管遇到什么难题，你都能从容应对。希望今天的分享能帮到你，以后咱们再遇到数学题，别害怕，先试试用抽象思维过滤一下，再用逻辑推理推一推，说不定你就会发现，原来数学也没那么难。好了，今天就聊到这儿，咱们下期再见。
