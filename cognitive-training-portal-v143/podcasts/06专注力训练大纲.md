# 青少年专注力训练实用指南

## 内容大纲

### 开场（1.5分钟）
上课时明明很努力在听，思绪却飘到了十万八千里外？做题做着做着就想起游戏？专注力差不是你态度问题，而是大脑需要训练。今天教你三招，让你的专注力像激光一样精准！

### 第一部分：冥想训练（2分钟）
别担心，冥想不是打坐，而是一种"大脑健身操"。

3分钟呼吸冥想：
1. 找一个舒服的坐姿
2. 闭上眼睛
3. 专注呼吸：吸气...呼气...
4. 如果走神了，温柔地把注意力拉回来
5. 每天练习3分钟，逐渐延长到10分钟

为什么有效？冥想训练的是"察觉走神→拉回来"的能力，这个能力迁移到学习上，就是专注力！

坚持练习一周，你会发现：走神的次数变少了，回到正轨的速度变快了。

### 第二部分：环境管理（2分钟）
手机管理：
- 学习时手机放到另一个房间
- 用"番茄钟APP"的锁屏功能
- 或者把手机交给父母保管

学习区域整理：
- 桌面只放当前需要的物品
- 无关的东西全部收走
- 保持桌面整洁

背景音乐选择：
- 适合：白噪音、自然音、轻柔钢琴
- 不适合：带歌词的流行歌、打游戏时的BGM

### 第三部分：注意力分段技巧（2分钟）
大脑工作90分钟后效率会大幅下降，所以要分段学习：

番茄工作法（之前讲过）：
- 25分钟专注 + 5分钟休息
- 4个番茄钟后长休息15-30分钟

休息时做什么？
- 站起来活动一下
- 喝口水
- 看看窗外远处
- 绝对不要看手机！

快速恢复专注力：
- 发现自己走神了，立刻做3次深呼吸
- 在心里默念"回来"，把注意力拉回来
- 不要责备自己，走神是正常的

### 总结（1分钟）
专注力训练计划：
- [ ] 每天3分钟呼吸冥想
- [ ] 学习时手机远离
- [ ] 用番茄钟管理时间
- [ ] 休息时真正休息

坚持21天，你会发现：
- 上课能跟上老师的节奏了
- 作业完成速度变快了
- 有时间做自己喜欢的事了

专注力是后天可以训练的，你的大脑比你想象的有潜力！
