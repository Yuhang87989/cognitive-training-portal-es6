# 青少年高效复习策略指南

## 内容大纲

### 开场（1.5分钟）
很多同学都有一个误区：复习就是一遍遍看书。其实，低效复习是最大的时间浪费。今天教你三招科学复习法，让你的学习效率翻倍！

### 第一部分：艾宾浩斯遗忘曲线（2分钟）
100多年前，德国心理学家艾宾浩斯发现：
- 学完20分钟：忘掉42%
- 1小时后：忘掉56%
- 1天后：忘掉66%
- 1周后：忘掉75%
- 1个月后：忘掉79%

好消息：忘掉是正常的！大脑不是硬盘，它的设计本来就会遗忘。关键是用对方法对抗遗忘。

### 第二部分：主动回忆法（2分钟）
被动阅读vs主动回忆，效果差3倍！

闭眼回忆法：
1. 看完一页书
2. 闭上眼睛
3. 试着回忆刚才看的内容
4. 能想起多少就写多少
5. 打开书对照，只看没记住的部分

这个方法一开始会很难受，但坚持一周，你会发现记忆效率大幅提升！

### 第三部分：间隔复习计划表（2分钟）
根据遗忘曲线制定复习时间表：

| 次数 | 时间点 | 复习内容 |
|------|--------|----------|
| 第1次 | 学完立即 | 全部内容 |
| 第2次 | 当天睡前 | 全部内容 |
| 第3次 | 第2天 | 重点部分 |
| 第4次 | 第4天 | 重点部分 |
| 第5次 | 第7天 | 难点部分 |
| 第6次 | 第15天 | 难点部分 |
| 第7次 | 第30天 | 全部内容 |

制作方法：在日历上标出复习日期，完成就打勾！

### 总结（1分钟）
复习策略清单：
- [ ] 课后当天复习
- [ ] 用主动回忆代替反复阅读
- [ ] 制作个人复习日历
- [ ] 坚持21天养成习惯

记住：复习一次顶自学三次！
