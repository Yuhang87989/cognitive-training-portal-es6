# Week 2 训练计划表

## 一、概述

### 1.1 训练目标
- **总体目标**：在Week 1基础上深化注意力与记忆力训练，引入**学霸学习方法专题模块**和**听课习惯培养**，帮助青少年建立高效学习系统
- **认知重点**：
  - **注意力**：提升多任务处理能力、冲突抑制、注意控制
  - **记忆力**：增强复杂信息整合、工作记忆刷新、记忆策略系统化应用
  - **学霸专题**：深度理解与知识建构、高效解题策略、考试冲刺技巧、长期学习规划
  - **听课习惯**：课前预习、课中专注、课后复习、笔记整理、提问互动

### 1.2 适用对象
- **年龄段**：12-16岁青少年
- **版本选择**：
  - **小学生版**：适合12-14岁，侧重基础认知训练，游戏化设计简单有趣
  - **初中版**：适合14-16岁，侧重高级认知训练，包含逻辑推理和多重任务
- **训练时长**：完全由用户自定义（建议15-30分钟）
- **难度控制**：手动调整（入门→专家五级）

### 1.3 训练原则
1. **每日坚持**：建议周一至周五训练，周末休息或复习
2. **循序渐进**：从Week 1基础向Week 2高级任务过渡
3. **趣味优先**：游戏化设计保持训练动机
4. **个性化适配**：根据用户偏好动态生成内容
5. **学用结合**：将学霸学习方法和听课习惯融入日常认知训练

|------|----------|----------|----------|-------------------|--------------|

**训练次数说明**：系统默认每天8次训练（即8个独立session），用户可自定义调整为1次、2次、3次等。每次训练包含注意力、记忆力、学习方法应用三个模块，时长按用户偏好分配。

### 2.2 学霸专题模块安排
- **频次**：每周安排2次学霸学习方法专题模块（Day 1和Day 3）
- **内容**：深度理解与知识建构、高效解题策略、考试冲刺技巧、长期学习规划
- **整合方式**：在当日的学习方法应用模块中系统讲解和练习

### 2.3 听课习惯培养安排
- **分散融入**：Day 2和Day 4的学习方法应用模块专门训练听课习惯
- **核心习惯**：课前预习（预读、问题标记）、课中专注（专注技巧、互动参与）、课后复习（及时复习、知识整理）、笔记整理（高效记录、重点标记）、提问互动（主动提问、讨论参与）
- **日常应用**：在每日注意力、记忆力训练中提醒应用相关听课习惯

## 三、每日训练计划（Week 2）

### 3.1 训练结构说明
每个训练日包含3个模块，时长完全由用户自定义（按训练次数分配）：
1. **注意力训练模块**（建议6-8分钟）
2. **记忆力训练模块**（建议6-8分钟）
3. **学习方法应用模块**（建议6-8分钟）——融入学霸专题或听课习惯

**版本差异**：
- **小学生版**：任务更简单直观，视觉引导强，即时反馈频繁，学习方法通过游戏化场景引入
- **初中版**：任务复杂度更高，包含逻辑推理和多重约束，学习方法侧重系统化应用和策略优化

**难度分级**：
- **入门级**（难度1）：充足反应时间，强引导，简单任务
- **初级**（难度2）：适中时间限制，少量干扰，基础复杂度
- **中级**（难度3）：时间压力适度，中等干扰，需策略应用
- **高级**（难度4）：严格时间限制，高干扰，复杂多任务
- **专家级**（难度5）：极限时间压力，多重干扰，需要高级认知控制
