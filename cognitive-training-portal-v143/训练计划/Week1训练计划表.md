# Week 1 训练计划表

## 一、概述

### 1.1 训练目标
- **总体目标**：建立每日训练习惯，掌握基础注意力与记忆力训练方法，并学习科学学习方法以提升学习效能
- **认知重点**：
  - **注意力**：提升视觉追踪能力、简单听觉注意
  - **记忆力**：增强图片记忆、短时词语复述
  - **综合应用与学习策略**：培养日常问题解决能力，学习并应用记忆宫殿、费曼技巧等科学学习方法

### 1.2 适用对象
- **年龄段**：12-16岁青少年
- **版本选择**：
  - **小学生版**：适合12-14岁，侧重基础认知训练，游戏化设计简单有趣
  - **初中版**：适合14-16岁，侧重高级认知训练，包含逻辑推理和多重任务
- **训练时长**：完全由用户自定义（建议10-30分钟）
- **难度控制**：手动调整（入门→专家五级）

### 1.3 训练原则
1. **每日坚持**：建议周一至周五训练，周末休息或复习
2. **循序渐进**：从简单任务开始，逐步增加挑战
3. **趣味优先**：游戏化设计保持训练动机
4. **个性化适配**：根据用户偏好动态生成内容

## 二、多用户支持说明

### 2.1 用户信息库概述
系统已建立SQLite数据库（`data/shared_state/users.db`），支持多用户独立训练。当前包含3个示例用户：
- **小明**：12岁，小学，小学生版，难度2，15分钟
- **小红**：14岁，初中，初中版，难度3，20分钟  
- **小刚**：13岁，初中，初中版，难度2，18分钟

### 2.2 新用户创建流程
当新用户首次使用系统时，系统将引导完成以下信息注册：
1. **姓名**：唯一标识（无需密码，简化登录）
2. **性别**：男/女/其他
3. **年龄**：6-18岁
4. **教育阶段**：小学/初中
5. **版本偏好**：小学生版/初中版
6. **每日训练时长**：完全自定义（分钟）
7. **偏好训练时间**：可选，如“18:00”

**示例交互**：
```
系统：检测到新用户，请输入姓名：[用户输入]
系统：请输入性别（男/女/其他）：[用户选择]
系统：请输入年龄（6-18岁）：[用户输入]
...
系统：用户【姓名】创建成功！开始你的训练之旅吧！
```

### 2.3 用户切换机制
- **快速切换**：通过“切换到[姓名]”命令或选择用户列表编号
- **智能推荐**：系统默认显示最近登录用户
- **进度保持**：每个用户的训练记录、积分、连续打卡独立存储

**示例交互**：
```
系统：当前用户【小明】，今日训练时长：15分钟
用户：切换到小红
系统：已切换到用户【小红】，当前难度级别：3，今日训练时长：20分钟
```

### 2.4 用户进度查看
系统提供多种进度查询方式：
- **即时查询**：训练后显示本次表现数据（反应时间、正确率、积分）
- **周度报告**：每周日生成个性化进展分析
- **排行榜**：查看家庭或班级内用户积分排名（可选功能）

## 三、每日训练计划（Week 1）

### 3.1 训练结构说明
每个训练日包含3个模块，时长完全由用户自定义：
1. **注意力训练模块**（建议5-7分钟）
2. **记忆力训练模块**（建议5-7分钟）
3. **综合应用与学习策略模块**（建议5-6分钟）

**版本差异**：
- **小学生版**：任务更简单直观，视觉引导强，即时反馈频繁，学习方法通过游戏化引入
- **初中版**：任务复杂度更高，包含逻辑推理，认知负荷适度增加，学习方法侧重系统化应用

- 积分奖励：每正确完成一项任务得10分

- 进度显示：训练条实时展示完成比例

## 四、游戏化设计体系

### 4.1 积分系统
- **基础积分**：每项任务正确完成获得10分
- **连击奖励**：连续正确3次额外获得5分
- **速度加成**：反应时间在前20%获得1.5倍积分
- **策略奖励**：自主使用有效策略获得额外5分

### 4.2 成就徽章
| 徽章名称 | 获取条件 | 激励作用 |
|----------|----------|----------|
| 初露锋芒 | 完成首次训练 | 建立初始成就感 |
| 记忆大师 | 单次记忆任务全部正确 | 鼓励记忆准确性 |
| 速度之星 | 反应时间进入前20% | 提升信息处理速度 |
| 全能王 | 单日三个模块均达优秀 | 促进综合能力发展 |
| 温故知新 | 完成复习训练 | 鼓励巩固与反思 |
| 坚持之星 | 连续打卡5天 | 培养训练习惯 |

### 4.3 挑战任务
- **每日挑战**：每天设置一个额外挑战任务（可选）
- **周度目标**：每周设定一个综合能力提升目标
- **排行榜**：家庭/班级内用户积分排名（保护隐私前提下）

### 4.4 视觉反馈
- **进度条**：实时显示训练完成比例
- **星级评价**：根据表现给予1-5星评价
- **数据可视化**：用图表展示进步趋势

## 五、难度控制指南

### 5.1 难度级别定义（手动控制）
| 级别 | 描述 | 适用阶段 |
|------|------|----------|
| **入门级** | 零基础，需要充分引导和反应时间 | 初次接触或建立信心 |
| **初级** | 有基础，适中挑战与成就感平衡 | 第1-2周主流难度 |
| **中级** | 熟练者，需要认知努力和策略应用 | 第3-8周逐步提升 |
| **高级** | 经验丰富，复杂任务和多重约束 | 第9周及以后挑战 |
| **专家级** | 追求极限，高认知负荷和时间压力 | 能力突出者专项训练 |

### 5.2 调整维度
用户可手动调整以下维度控制难度：
1. **任务复杂度**：信息量、约束条件数量
2. **反应时间**：从宽松到严格的时间限制
3. **干扰强度**：干扰项数量和质量
4. **认知负荷**：同时处理的信息类型和数量
5. **记忆负荷**：记忆项目数量、保持时间

### 5.3 调整建议
- **初始设置**：建议从入门或初级开始
- **提升时机**：连续3天正确率>90%可尝试提升一级
- **降低时机**：连续2天正确率<70%建议降低难度
- **版本切换**：如觉得当前版本太简单/难，可切换另一版本

### 5.4 系统智能推荐
基于用户历史表现，系统将提供难度建议：
1. **表现分析**：过去7天平均正确率、反应时间
2. **进步趋势**：连续训练中的能力提升曲线
3. **主观反馈**：用户自评的难度感受和训练负荷
4. **最终决策**：用户手动确认调整

## 六、周度复盘机制

### 6.1 数据收集流程
每周日系统自动执行以下数据收集：

**数据源**：
1. **训练记录表**（`training_records`）：查询本周所有训练记录
2. **用户进度表**（`user_progress`）：获取累计积分、连续打卡等统计
3. **用户信息表**（`users`）：获取用户配置偏好

**关键指标**：
- **注意力指标**：平均反应时间、持续注意任务正确率、抗干扰能力
- **记忆力指标**：工作记忆广度、信息提取速度、记忆准确率
- **综合指标**：积分增长、连续打卡天数、训练时长稳定性

### 6.2 多用户数据处理
对于家庭或机构多用户场景，系统将：
1. **独立分析**：为每个活跃用户生成个性化报告
2. **对比分析**：在用户同意前提下，提供同龄人表现对比（匿名化）
3. **群体统计**：管理者可查看整体训练参与度和进步趋势

### 6.3 复盘报告内容
《Week 1训练复盘报告》包含以下部分：

**1. 本周表现概览**
- 训练天数/总天数
- 总积分及增长
- 平均正确率和反应时间

**2. 各能力维度进展**
- 注意力：视觉追踪、听觉注意、抗干扰能力变化
- 记忆力：图片记忆、词语记忆、工作记忆广度变化
- 综合应用：问题解决能力、计划制定能力评估

**3. 游戏化成就汇总**
- 获得的徽章列表
- 挑战任务完成情况
- 积分排名（如启用）

**4. 个性化建议**
- 基于表现的难度调整建议
- 重点加强的能力方向
- Week 2训练计划预览

**5. 用户反馈收集**
- 训练感受评分（1-5分）
- 最喜欢的训练模块
- 希望改进的方面

### 6.4 复盘执行时间
- **自动触发**：每周日晚上8点（可配置）
- **手动触发**：用户可随时请求生成报告
- **推送通知**：报告生成后通过APP/浏览器通知用户

## 七、后续自动化流程

### 7.1 每日训练内容生成
基于用户信息库配置，系统每日动态生成训练内容：

**生成逻辑**：
1. **读取用户配置**：版本偏好、难度级别、训练时长
2. **参考训练计划**：根据Week X Day Y确定当日训练主题
3. **个性化适配**：
   - 调整任务复杂度匹配难度级别
   - 分配时间比例匹配训练时长
   - 选择游戏形式匹配版本特点
4. **内容输出**：生成结构化训练模块（文本描述+交互指导）

**示例流程**：
```
输入：用户ID=2（小红，初中版，难度3，20分钟）
输出：
1. 注意力模块（7分钟）：高级注意控制挑战
2. 记忆力模块（7分钟）：工作记忆强化营  
3. 综合应用模块（6分钟）：逻辑推理实战
```

### 7.2 多用户并发支持
系统支持多用户同时训练，数据隔离机制：
1. **会话隔离**：每个训练会话关联特定用户ID
2. **进度独立**：训练记录、积分、成就独立存储
3. **快速切换**：支持训练中途切换用户身份

### 7.3 数据同步与备份
- **实时保存**：每次训练后立即保存记录
- **定期备份**：每周自动备份数据库
- **异常恢复**：网络异常时本地暂存，恢复后同步
