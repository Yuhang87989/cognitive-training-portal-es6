# Week 7 训练计划表（4月28日-5月4日）
## 元认知深化与综合能力迁移周

### 一、周度核心目标
**总体目标**：在Week 6学科整合与自主学习能力建立的基础上，开启"元认知深化+习惯固化+能力迁移"的三轨并行模式，重点实现从"知道方法"到"自动化运用"的跨越，建立可持续的学习力内化机制。

**能力侧重**：
1. **元认知深化**：从"初步意识"升级到"主动调控"，实现元认知能力的自动化运行
2. **自主学习习惯固化**：计划-执行-复盘闭环成为自然的学习节奏
3. **学科整合能力升级**：从简单问题到复杂真实场景的应用
4. **综合能力迁移**：将训练中习得的方法迁移到学校真实学习场景
5. **学霸方法内化**：深度理解→解题策略→考试技巧的完整闭环自动化

**预期成果**：
- 能够主动监控和调节自己的学习过程，元认知成为学习习惯
- 自主学习闭环稳定运行，自我管理效率显著提升
- 能够独立解决中等复杂度的跨学科问题
- 训练方法与学校学习形成良性互动，相互促进
- 建立个人专属的"学习操作系统"，实现学习自动化

#### Day 1（4月28日，周二）：元认知深化 - 从"意识到"到"调控好"
**核心目标**：深化元认知能力，实现从"知道我该监控"到"能够主动调控"的升级，建立元认知自动化运行机制

**训练主题**：成为自己的思维教练
- **入门路径（邱宇菲/难度1）**：
  - "思维健身房"：元认知练习如同体育锻炼，需要刻意训练
  - "自我教练三步法"：观察→判断→调节的快速运行
  - "学习状态仪表盘"：实时监控自己的注意力、理解力、动力状态
  - 训练重点：元认知反应速度提升、调控策略丰富化、自动运行意识建立

**元认知深化三步法**：
1. **快速观察**：学习中"扫描"自己的状态（5秒内判断）
2. **即时判断**：识别状态是否需要调整（好/一般/差）
3. **快速调节**：选择合适的调节策略并执行（休息/换方法/换节奏）

**元认知调节策略库**：
- 注意力不集中：深呼吸3次→重新聚焦→设定小目标
- 理解困难：换个角度→求助工具→暂时跳过
- 动力不足：想想目标→奖励自己→改变环境

**学霸方法专题**：
- 考试中的元认知：读题时判断难度、分配时间时监控状态、遇到难题时决策
- 错题本与元认知：分析"当时的我是怎么想的"比"正确答案是什么"更重要

#### Day 2（4月29日，周三）：自主学习习惯固化 - 让计划成为自然
**核心目标**：将自主学习流程内化为自然习惯，实现"不用刻意想就能做好"的目标，形成稳定的学习节奏

**训练主题**：学习节奏的自动化运行
- **入门路径（邱宇菲/难度1）**：
  - "习惯的力量"：了解习惯形成的科学原理（小步快跑、即时反馈、循环强化）
  - "我的学习仪式"：建立固定的学习开始和结束仪式
  - "21天习惯追踪"：记录自主学习习惯的形成过程
  - 训练重点：习惯回路建立、仪式感培养、习惯追踪与调整

**自主学习习惯回路**：
1. **提示（Cue）**：学习时间到了/看到学习环境/想起今日任务
2. **常规（Routine）**：打开任务→制定计划→开始执行→记录复盘
3. **奖励（Reward）**：完成任务成就感/获得积分/得到鼓励

**习惯固化工具**：
- "学习启动清单"：打开书包→摆好工具→打开任务→深呼吸
- "学习结束仪式"：整理笔记→打卡记录→计划明天→收拾书包
- "习惯追踪表"：每日记录习惯执行情况，周末分析趋势

**听课习惯培养**：
- 课堂中的元认知：老师在讲什么？我理解了吗？要不要记下来？
- 笔记的元认知意义：记笔记是帮助自己思考，不是机械抄写

#### Day 3（4月30日，周四）：能力迁移专项 - 从训练场到真实战场
**核心目标**：将训练中习得的思维方法和学习策略迁移到学校真实学习场景，实现"学以致用"的目标

**训练主题**：把训练变成超能力
- **入门路径（邱宇菲/难度1）**：
  - "迁移小侦探"：发现学校学习和训练中的相似之处
  - "方法对接练习"：训练方法↔学校场景的对应练习
  - "真实问题挑战"：用学到的思维方法解决学校作业中的问题
  - 训练重点：迁移意识建立、方法对接能力、实践应用信心

**迁移练习四步法**：
1. **识别**：这个问题像训练中的哪个类型？
2. **回忆**：我当时是怎么解决的？
3. **调整**：这次有什么不同？需要怎么调整？
4. **应用**：执行并验证效果

**迁移场景对照**：
| 训练内容 | 学校场景 | 迁移要点 |
|----------|----------|----------|
| 逻辑推理 | 数学证明题 | 步骤清晰、有根有据 |
| 模型构建 | 物理应用题 | 提取关键、简化问题 |
| 自主学习 | 作业规划 | 目标分解、优先级排序 |
| 元认知 | 考试策略 | 时间分配、状态监控 |
| 复盘反思 | 错题分析 | 原因分析、策略调整 |

**元认知在迁移中的作用**：
- 监控迁移过程：我的方法用对了吗？
- 评估迁移效果：这次迁移成功吗？为什么？
- 优化迁移策略：下次如何做得更好？

### 五、评估机制与质量标准
**评估维度**：

1. **元认知深化维度**：
   - 监控速度：从"意识到问题"到"开始调节"的时间
   - 调控有效性：调节策略的选择准确性和执行效果
   - 自动化程度：元认知运行的"无意识化"程度

2. **习惯固化维度**：
   - 习惯回路完整性：提示-常规-奖励的完整形成
   - 执行稳定性：习惯执行的一致性和频率
   - 仪式感程度：开始和结束仪式的固定程度

3. **能力迁移维度**：
   - 迁移意识：主动识别迁移机会的意识
   - 迁移方法：方法对接和调整的能力
   - 迁移效果：迁移后的实际应用效果

4. **综合学习力维度**：
   - 能力整合程度：多种能力的协同运用能力
   - 进化速度：学习能力的持续提升趋势
   - 迁移潜力：从训练到学校的正向迁移效果

**评估方法**：
- **过程性评估**：每日训练中的实时反馈与指导
- **习惯追踪评估**：习惯形成进度可视化追踪
- **迁移效果评估**：学校学习中的实际应用表现
- **总结性评估**：Week 7复盘报告的综合分析

**质量标准**：
- **优秀**：元认知高度自动化，习惯稳定运行，迁移意识强烈，综合能力突出
- **良好**：元认知基本自动化，习惯初步形成，迁移意识建立，综合能力较强
- **基础**：元认知有意识，习惯开始形成，需要强化迁移练习
- **待加强**：元认知意识薄弱，需要从习惯养成重新建立
