### 三、数据同步优化 (V112)

#### 统一存储键
- `cognitive_training_v50`

#### 统一数据结构
```javascript
{
    version: 50,
    users: [],           // 用户数据
    currentUser: null,   // 当前用户
    userVideos: [],      // 用户上传视频
    podcastPositions: {},// 播客播放位置
    mediaPositions: {},  // 媒体播放位置
    wrongQuestions: [],  // 错题本
    gameRecords: {},     // 游戏记录
    trainingProgress: {},// 训练进度
    lastSyncTime: ''     // 最后同步时间
}
```

#### 新增API函数
- `getDefaultData()` - 获取默认数据模板
- `getModuleData(module)` - 获取特定模块数据
- `saveModuleData(module, value)` - 保存特定模块数据
